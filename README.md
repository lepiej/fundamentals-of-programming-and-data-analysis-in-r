# Fundamentals of Programming and Data Analysis in R

Materials and exercises completed during the 1st semester of the MSc in Investment and Data Analytics at the University of Warsaw.
The course introduces the basics of programming and data analysis in R, focusing on coding logic, data manipulation, visualization, and simple statistical methods.
This repository includes scripts and notebooks exploring:
- Core R programming concepts
- Data cleaning and transformation
- Exploratory data analysis and visualization
- Basic statistical analysis
